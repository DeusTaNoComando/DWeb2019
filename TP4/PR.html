Abstract:

Este quarto trabalho prático divide-se em duas partes, inicialmente, o objetivo era a transformação de um documento XML longo em vários documentos XML em que cada um destes era composto apenas por um arqueossitio, posteriormente é usada uma stylesheet para transformar o XML em HTML.

A segunda parte do trabalho tinha como principal objetivo a criação, em nodeJS, de um servidor onde passando apenas como argumento o número do arqueossitio, o documento XML deste apareceria no ecrã. Portanto através do link: localhost:7777/ Nº do documento.

Deliverables:

  • XML do Project Record
  • HTML do Project Record
  • StyleSheet dos arqueossitios
  • Stylesheet para cada arqueossitio
  • Código do servidor
  • Pasta do ficheiro dos documentos XML de cada arqueossitio
